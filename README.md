# MNIST Flask demo 

For education purposes only.
The model is the simple 2-hidden layer neural network, created using vectorized numpy network implementation
[MLP MNIST](https://github.com/alm4z/mlp-mnist). 

## Running
1. Connect to github codespace or clone the repo in local.
2. Build and run the docker image using command - 

    ```docker-compose up --build```
3. When prompted open the application URL in browser.
